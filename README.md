## About

This is a **community-fixed fork** of [Harmony Music](https://github.com/anandnet/Harmony-Music) by [anandnet](https://github.com/anandnet) (3,000+ stars). The original project is a cross-platform music streaming app built with **Flutter** for Android, Windows, and Linux.

Since the original is no longer maintained, this fork keeps the app functional by fixing critical bugs and compatibility issues with current YouTube/Piped backends and newer Android versions.

## FAQ

**Why is Harmony Music search not working?**
The original app is unmaintained. YouTube and Piped API endpoints have changed, breaking search. The fixed APK updates these endpoints.

**Where can I download the fixed Harmony Music APK?**
Download from [harmonymusic.space-z.ai](https://harmonymusic.space-z.ai/).

**Is this Harmony Music APK safe?**
Yes. This is an open-source fork under GPL v3.0 license.

**Does it work on Android 14?**
Yes. All Android 13+ compatibility issues are resolved.
